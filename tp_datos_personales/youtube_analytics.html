Procesamiento de datos:
  • Takeout de youtube de 3 cuentas distintas, ver Dataset Original
  • Parsing del html original, agarrando de cada video el nombre, el uploader, y la fecha
  • Por cada video, se le pega a la API de youtube: https://www.googleapis.com/youtube/v3/videos?part=snippet&id=muewMnk8o5U&key=APIKEY, con cada request a la API, se consiga informacion extra del video: la category id, la lista de tags
  • Dado que la API de youtube es gratis, pero tiene una quota diaria, tuve que crear 4 cuentas de youtube, y crear una api key con cada una, tuve que hacer un proceso de mergeo de datos tambien
  • Usando los tags, se creo el wordcloud desde python
  • Para importar a tableau, tuve que sacar los tags del dataset, ya que al ser una lista, le traia problemas al import de tableau
  • Aca se puede ver todo el codigo que implica el proceso de scrapping y procesamiento de datos
